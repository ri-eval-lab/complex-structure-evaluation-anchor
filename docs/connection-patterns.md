# Connection patterns / 接続パターンについて

方式名『複素構造評価』は、既存手法等、既存出力、改善結果、運用出力、判断材料、研究結果、検証結果等と接続して用いられる場合があります。

本ページでいう「接続パターン」とは、これらの前段材料を、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』または方式名『複素構造評価』を媒介して、説明、共有、比較、記録、検証、既存様式への記入、既存手法への再接続、改善・運用サイクルへの戻し、または次の判断材料へつなぐ流れを指します。

## Existing methods and related materials / 既存手法等の範囲

本ページでいう「既存手法等 / existing methods and related materials」とは、既に用いられている説明テンプレート、思考・構造化手法、分析手法、評価手法、意思決定支援の枠組み、改善・品質・運用管理の手法、既存様式、既存出力、運用資料、研究結果、検証結果、判断材料等を広く指します。

ここでいう「手法」には、たとえば次のような類型を含みます。

- 表現・伝達の型 / communication methods
  - 例：PREP法、結論先行型等

- 思考・構造化の型 / reasoning and structuring methods
  - 例：MECE、ピラミッド原則、ロジックツリー等

- 分析・評価・意思決定支援の枠組み / analysis, evaluation, and decision-support methods
  - 例：SWOT、KPI/OKR、リスクマトリクス、意思決定マトリクス等

- 改善・品質・運用管理の手法 / improvement, quality, and operations methods
  - 例：PDCA、なぜなぜ分析、カイゼン等

また、「既存手法等」には、上記の手法だけでなく、報告書、評価票、会議メモ、監査記録、改善報告、研究結果、検証結果、判断材料など、既に作成された出力や運用資料も含みます。

上記は例示であり、接続可能な対象を限定するものではありません。

## 接続類型

接続パターンは、単発の「入力元 → 本方式 → 最終出力」に限られません。入力元、媒介層、出力先または次工程、既存手法への再接続、次回入力、改善サイクル等を含む、開いた接続として扱います。

代表的には、次のような接続類型が想定されます。

### 表現整理型

表現・伝達の型、説明材料、要点出し材料等を、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』で整え、説明文、報告、共有文等へ戻す接続です。

例：

表現・伝達の型  
→ 表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』  
→ 説明文・報告・共有文

### 表現整理から評価へ上げる型

表現整理済みの文面や判断材料を、必要に応じて方式名『複素構造評価』へ接続し、比較資料、記録、検証用メモ等へ上げる接続です。

例：

表現整理済みの文面・判断材料  
→ 表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』  
→ 必要に応じて方式名『複素構造評価』  
→ 比較資料・記録・検証用メモ

### 構造化材料の評価接続型

構造化された材料、分析結果、判断材料等を、方式名『複素構造評価』で比較可能な形へ整える接続です。

例：

構造化された材料・分析結果・判断材料  
→ 方式名『複素構造評価』  
→ 比較可能な評価出力・記録・判断材料

### 改善・運用出力接続型

改善結果、運用出力、管理資料、記録資料等を、方式名『複素構造評価』で同じ読み方にそろえる接続です。

例：

改善結果・運用出力・管理資料・記録資料  
→ 方式名『複素構造評価』  
→ 見直し用の整理・改善判断・比較資料

### 再接続型

既存手法の出力、既存様式、既存業務出力等を、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』または方式名『複素構造評価』を媒介して、別の既存手法、既存様式、評価会議、次工程等へ接続する型です。

例：

既存手法Aまたは既存出力  
→ 表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』または方式名『複素構造評価』  
→ 既存手法B・既存様式・評価会議・次工程

### 表現へ戻す型

方式名『複素構造評価』で整理した結果を、表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』を通じて、日常文面、説明文、報告、共有文等へ戻す接続です。

例：

方式名『複素構造評価』で整理した結果  
→ 表現整理フレーム『ＲＩＶ－ＴＨＥＴＡ』  
→ 日常文面・説明文・報告・共有文

本節に示す接続類型は、代表的な読み方を示すものであり、個別の業務、研究、実用性検証、説明、共有、比較、記録、改善サイクル等で生じる派生パターンをすべて閉じた形で列挙するものではありません。
